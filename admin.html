⚙️ System Settings

Browser-only settings (token + admin password).

Setting this GitHub PAT will add every submitted customer form to bookings.csv instantly (status = Pending). You won't need to add entries manually. WhatsApp will still open as usual.

Setup:

  1. Generate a fine-grained PAT: github.com/settings/personal-access-tokens/new
  2. Repository: BRIJBHAN-SINGH234/call4all
  3. Permissions → Contents → Read and write
  4. Expiration: 1 year (suggested)
  5. Copy the token, paste it here → Save Settings
⚠️ Security: This token is stored in your browser's localStorage (in this browser only). It is also exposed to every site visitor (since browser code can read it). Therefore, restrict the PAT to only the Contents: Read & Write permission.

If you set a new password, it will be used from the next login onwards.
